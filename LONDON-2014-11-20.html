Discussion

  • PostGIS & Postgres utilities

    • pgtune
      • Suggest sensible config based on memory, cores etc.
    • pgbench
      • Benchmark SQL
    • csvkit
      • Manipulate CSV incl. loading into PostgreSQL
    • mdb2postgresql
      • Load MDB into PostgreSQL
    • pl/pgsql debugger
      • pgAdmin plugin for debugging PL/SQL
    • pgAgent
      • A job scheduling agent for PostgreSQL, capable of running multi-step batch/shell and SQL tasks on complex schedules
    • Pyrseas
      • Database migrations for PostgreSQL
    • GeoTools
      • Access to PostGIS data in Java (used by GeoServer)
    • ppygis
      • Adds support for PostGIS geometry to psycopg
    • pgRouting
      • Extends the PostGIS / PostgreSQL geospatial database to provide geospatial routing functionality. Articles
